# ABSTEP_Start-1.0.0
Antigua and Barbuda Skills Training and Empowerment Programme (ABSTEP) software built by  Stevon C.J. Gordon, October 2014, for the Antigua and Barbuda Labour Department.

The four main entries used to calculate the employability index are: Educational Level(EL), Income Level(IL), Unemployment Length(UL) and Work Experience(WE). Based on a labour criteria provided by the Department, the software provides a score for each variable; then the scores are summed to give the employability index of the beneficiary.

The employability index determines if the beneficiary is eligible for the ABSTEP Training Programme or Temporary Employment Programme.
Beneficiaries with a score of 50 and above are eligible for ABSTEP Training Programme.
Beneficiaries with a score below 50 are eligible for ABSTEP Temporary Employability Programme.

Instruction:
To login to the software, username: abstep and the password: admin.

The user must enter the relevant information in the fields provided: 
Beneficiary's name, 
Beneficiary's social security number, 
Education level (A - F), 
Income level, 
Unemployment length,
Beneficiary's age,
Number of Weeks of Contribution.

All entries must be made before the Employability Index can be calculated.
If entries are incomplete the user will be reminded to make sure all entries are made.

To make the next entry click on the 'Enter Next Beneficiary' button. The previous entry will be recorded by the software, the total numbers entries made will be be shown below at 'Number of entries made'.
 
To view the entries click on the 'Show Document' button; a pop-up text editor will appear to show the list of entries that was made. The list can also be updated via the text editor. Click on the 'Read ...' button to view the entries; click on the 'Write ...' button to update the entry.
